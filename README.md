# Podman Lab: Panduan Lengkap dari Dasar ke Advanced

Selamat datang di materi praktikum Podman. Materi ini dirancang untuk membantu Anda memahami penggunaan Podman sebagai alternatif Docker yang aman, ringan, dan modern.

## Daftar Modul

1.  **[1. Instalasi & Pengenalan Podman](./lab/01-instalasi-pengenalan.md)**
    *   Pengenalan arsitektur dan cara instalasi.
2.  **[2. Manajemen Image](./lab/02-manajemen-image.md)**
    *   Bermain dengan image, registry, dan tagging.
3.  **[3. Manajemen Container](./lab/03-manajemen-container.md)**
    *   Mengelola siklus hidup container dan limitasi resource.
4.  **[4. Build Custom Image](./lab/04-build-image.md)**
    *   Membuat image sendiri dengan Dockerfile dan Multi-stage build.
5.  **[5. Volume & Persistent Storage](./lab/05-volume-storage.md)**
    *   Penyimpanan data permanen dan izin di rootless mode.
6.  **[6. Networking](./lab/06-networking.md)**
    *   Konfigurasi jaringan antar container.
7.  **[7. Rootless Container & Security](./lab/07-rootless-security.md)**
    *   Keamanan container dan mode rootless.
8.  **[8. Pod (Konsep Unik Podman)](./lab/08-pod-konsep.md)**
    *   Mengelompokkan container dalam Pod.
9.  **[9. Podman Compose](./lab/09-podman-compose.md)**
    *   Orkestrasi aplikasi multi-service.
10. **[10. Generate Kubernetes YAML & Systemd](./lab/10-kube-systemd.md)**
    *   Integrasi dengan Kubernetes dan autostart service.

---
Materi ini dibuat untuk mempermudah transisi dari Docker ke Podman serta memaksimalkan fitur keamanan container.
